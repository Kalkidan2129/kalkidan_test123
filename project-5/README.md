
# Walmart Store Sales

## Project Overview

This project involved creating a comprehensive Power BI dashboard to analyze year-over-year sales data from 45 Walmart stores across the US between 2010 and 2012. The analysis highlighted sales impacts during the post-recession period, enabling data-driven insights into retail performance. Key data modeling and DAX measures were developed to support dynamic sales reporting.

---

## Business Problem

Walmart stores faced fluctuating sales during the recovery period following the 2008 recession, impacting revenue and operational planning. Understanding sales trends and external factors was critical to optimizing store performance and inventory management. The challenge was to consolidate and visualize multi-year sales data to identify patterns and inform business strategy.

---

## Objective

- Analyze year-over-year sales trends across multiple Walmart stores
- Incorporate external factors like holidays and unemployment rates into sales analysis
- Develop an interactive Power BI dashboard for actionable retail insights

---

## Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
- Excel (for initial data handling)
- CSV data files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create calendar and refresh date tables for time intelligence
- Establish relationships between sales and calendar tables
- Develop conditional columns for holidays and unemployment rates
- Build DAX measures to calculate total, average, and year-over-year sales
- Design and customize the dashboard with logos and titles
- Apply sorting and formatting for clear data presentation
- Deploy the dashboard to Power BI Cloud for sharing

---

## Key Insights

- Sales trends showed variability influenced by holiday weeks and economic conditions post-recession
- Certain Walmart stores experienced distinct year-over-year sales growth patterns
- Unemployment rate segments correlated with fluctuations in weekly sales performance
- Custom time intelligence measures enabled clear visualization of monthly and yearly sales comparisons
