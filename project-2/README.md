
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a Power BI dashboard to analyze the effect of pilot awareness on wildlife strike incidents in aviation. It compared outcomes between well-informed warned pilots and uninformed pilots to assess damage control and passenger safety. Data modeling and SQL Server were utilized to build a comprehensive data mart for reporting.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and cause costly damages. There is a need to understand how pilot awareness and warnings influence these incidents. Improving pilot information could reduce damage costs and enhance passenger safety during flights.

---

## Objective

- Create a data mart from wildlife strike data using SQL Server
- Develop key measures to quantify warned pilot impact
- Build an interactive Power BI dashboard for visual insights

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL Scripts
- Data Modeling
- Reporting

---

## Project Workflow

- Load wildlife strike data into SQL Server to create a data mart
- Import and prepare data in Power BI
- Clean and model data, handling null values
- Create measures to calculate warned pilot percentages
- Design and format dashboard with maps, legends, and logos

---

## Key Insights

- Well-informed warned pilots correlate with reduced damage costs from wildlife strikes
- Pilot awareness significantly impacts flight safety during wildlife encounters
- Geographical variations exist in warned pilot percentages by state
- Effective data visualization aids in communicating safety insights to stakeholders
