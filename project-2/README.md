
# Global Superstore Sales Dashboard

## Project Overview

This project involved creating an interactive Power BI dashboard to visualize global superstore sales data segmented by country, customer segment, category, and subcategory. It enabled stakeholders to analyze sales trends and performance metrics effectively during the shift to online shopping accelerated by the pandemic. The dashboard facilitated data-driven decisions by consolidating multiple data points into a cohesive visual interface.

---

## Business Problem

The global pandemic drastically changed consumer shopping behavior, increasing the reliance on e-commerce channels. Retailers needed a comprehensive tool to monitor sales performance across regions and product categories to adapt strategies quickly. Without consolidated insights, identifying growth opportunities and addressing declining segments was challenging.

---

## Objective

- Create a dynamic dashboard visualizing sales by country, segment, category, and subcategory
- Enable tracking of key sales performance indicators with interactive visuals
- Support data-driven decision-making to respond to changing consumer behaviors

---

## Tools & Technologies

- Power BI
- Excel
- DAX
- Data Modeling
- Data Visualization

---

## Project Workflow

- Gather and model sales data from multiple sources
- Design and build interactive Power BI visuals and KPIs
- Implement DAX calculations for dynamic metrics
- Create filters and slicers for user-driven analysis
- Publish and refine dashboard based on stakeholder feedback

---

## Key Insights

- Sales performance varies significantly by country and customer segment
- Certain product categories and subcategories outperformed others during the pandemic
- E-commerce sales increased due to safety and convenience factors
- Interactive visuals reveal temporal sales trends and highlight growth opportunities
