
# Walmart Store Sales

## Project Overview

This project involved creating a comprehensive Power BI dashboard to analyze year-over-year sales data from 45 Walmart stores across the US between 2010 and 2012. The analysis provided insights into sales performance during the recovery period following the 2008 recession. Key data modeling, DAX measures, and visualizations were developed to support business decision-making.

---

## Business Problem

Walmart stores, like other discount retailers, faced sales challenges during the post-recession recovery period. Understanding sales trends and factors affecting performance was critical to optimize store operations and marketing strategies. The project aimed to identify sales patterns and external influences such as holidays and unemployment rates.

---

## Objective

- Import and model Walmart sales data in Power BI
- Create meaningful DAX measures to analyze sales trends
- Develop an interactive dashboard to visualize year-over-year sales performance

---

## Tools & Technologies

- Power BI Desktop
- DAX
- Excel
- Power Query Editor
- CSV Data Files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create calendar and refresh date tables using DAX
- Establish relationships between sales and calendar tables
- Develop conditional columns for holidays and unemployment rates
- Build DAX measures and design the final interactive dashboard

---

## Key Insights

- Sales trends showed fluctuations influenced by holiday weeks and unemployment rates
- Year-over-year sales growth was measurable despite the post-recession economic challenges
- Custom sorting of calendar data improved the accuracy of time-based analysis
- DAX measures enabled detailed monthly and average sales comparisons across stores
