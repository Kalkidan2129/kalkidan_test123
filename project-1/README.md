
# Walmart Store Sales

## Project Overview

This project involved creating a comprehensive Power BI dashboard to analyze year-over-year sales data from 45 Walmart stores across the US between 2010 and 2012. It included data import, modeling, DAX measures, and visualizations to uncover sales trends during the post-2008 recession recovery period.

---

## Business Problem

Walmart stores, like other discount retailers, faced sales challenges emerging from the 2008 recession. Understanding sales performance trends was critical to identifying recovery patterns and strategic opportunities.

---

## Objective

- Import and model Walmart sales data in Power BI
- Create calculated columns and DAX measures for sales analysis
- Develop a dynamic dashboard to visualize year-over-year sales trends

---

## Tools & Technologies

- Power BI Desktop
- DAX
- Excel
- Power Query Editor
- CSV Data Files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create calendar and refresh date tables for time intelligence
- Establish relationships between sales and calendar tables
- Build conditional columns and sorting for enhanced analysis
- Develop DAX measures and design the final sales dashboard

---

## Key Insights

- Sales trends showed fluctuations influenced by the post-recession economic environment
- Holiday weeks and unemployment rates were key factors affecting sales performance
- Month-over-month and year-over-year sales comparisons highlighted recovery progress
- Custom sorting and conditional columns improved clarity in sales pattern analysis
