
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a comprehensive Power BI dashboard to analyze the impact of pilot awareness on wildlife strike incidents. It compared outcomes between well-informed and uninformed pilots, focusing on damage costs and passenger safety. The solution leveraged SQL Server for data management and advanced KPI visualizations to support decision-making.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and result in costly damages. There is a need to understand how pilot awareness and timely warnings can mitigate these risks and reduce repair costs. Improving this awareness can enhance passenger safety and operational efficiency.

---

## Objective

- Analyze the effect of pilot warnings on wildlife strike damage costs
- Develop KPIs to measure pilot awareness and incident outcomes
- Create an interactive dashboard for data-driven decision support

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling

---

## Project Workflow

- Load and prepare wildlife strike data using SQL Server
- Import and model data within Power BI
- Create slicers for time periods, flight phases, and regions
- Develop KPI cards for pilot warnings, damage, and costs
- Design and finalize interactive dashboard for insights

---

## Key Insights

- Well-informed warned pilots correlate with reduced damage costs
- Timely warnings improve passenger safety during wildlife strikes
- Year-over-year trends reveal changes in pilot warning effectiveness
- Damage and repair costs vary significantly by flight phase and region
