
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a comprehensive dashboard to analyze the impact of pilot awareness on wildlife strikes in aviation. It compares outcomes between well-informed and uninformed pilots to assess damage costs and passenger safety. The solution leverages SQL Server and Power BI for data integration and visualization.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and result in costly damages. Lack of pilot awareness can increase the severity of these incidents, impacting operational costs and passenger safety. The project addresses the need to quantify and mitigate these risks through data-driven insights.

---

## Objective

- Analyze the effect of pilot warnings on wildlife strike damage costs
- Create interactive visualizations to monitor incident trends and KPIs
- Provide actionable insights to improve flight safety and reduce repair expenses

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling

---

## Project Workflow

- Load and prepare wildlife strike data using SQL scripts
- Import and model data in Power BI
- Create slicers and filters for dynamic data exploration
- Develop KPI cards to track pilot warning effectiveness and damage costs
- Design and finalize dashboard with logos, titles, and visual elements

---

## Key Insights

- Well-informed pilots significantly reduce damage costs from wildlife strikes
- Higher percentage of warned pilots correlates with fewer injuries and fatalities
- Damage repair costs decrease when pilots receive timely wildlife warnings
- Interactive slicers reveal trends by time period, flight phase, and region
