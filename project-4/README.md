
# Walmart Store Sales

## Project Overview

This project involved creating a comprehensive Power BI dashboard to analyze sales data from 45 Walmart stores in the US between 2010 and 2012. It focused on assessing the impact of the post-recession period on store sales using data modeling, DAX measures, and conditional columns.

---

## Business Problem

Walmart stores faced fluctuating sales during the recovery period following the 2008 recession, impacting revenue and operational decisions. Understanding sales trends and external factors was critical to optimize store performance and strategy.

---

## Objective

- Analyze year-over-year sales trends across multiple Walmart stores
- Incorporate external factors like holidays and unemployment rates into sales analysis
- Create an interactive Power BI dashboard for actionable business insights

---

## Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
- Excel (for data preparation)
- CSV data files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create calendar and refresh date tables for time intelligence
- Establish relationships between sales and calendar tables
- Add conditional columns for holiday flags and unemployment rates
- Develop DAX measures for sales aggregation and year-over-year analysis

---

## Key Insights

- Sales trends showed variability influenced by economic recovery post-2008 recession
- Holiday periods had measurable impacts on weekly sales performance
- Unemployment rate segments correlated with fluctuations in store sales
- Custom sorting and calendar integration enhanced temporal sales analysis
