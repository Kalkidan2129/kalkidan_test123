
# Revenue Forecasting and Performance Analysis of Vodafone Qatar (2021–2023)

## Project Overview

This project transformed Vodafone Qatar’s raw telecom data into a consolidated, machine-learning-ready dataset using NexusMax, Python, and Microsoft Fabric. It enabled advanced revenue forecasting, churn analysis, and performance tracking through interactive dashboards, supporting data-driven decisions for finance and strategy teams. The solution uncovered key patterns in customer behavior and network performance to optimize budget planning and reduce revenue risk.

---

## Business Problem

Vodafone Qatar faced challenges in understanding regional and segment revenue performance due to fragmented data and fluctuating subscriber churn. This lack of a unified view hindered accurate forecasting and confident decision-making for finance and strategy teams. The project addressed the need for consolidated insights to improve revenue predictability and operational planning.

---

## Objective

- Create a unified, clean, and ML-ready telecom dataset for analysis
- Improve quarterly revenue forecasting accuracy to optimize investments
- Reduce churn in key customer segments to protect recurring revenue

---

## Tools & Technologies

- NexusMax
- Python
- Microsoft Fabric
- Power BI
- Pandas
- Numtra

---

## Project Workflow

- Ingest and clean raw Vodafone Qatar telecom data
- Engineer and validate key performance metrics and time tables
- Develop supervised regression models for revenue forecasting
- Build interactive Power BI dashboards for performance tracking
- Deploy unified ML-ready dataset for strategic decision-making

---

## Key Insights

- Total revenue increased 14% quarter-over-quarter, driven mainly by Doha region
- Churn rate for high-value prepaid customers rose by 0.3% QoQ, impacting revenue stability
- Network availability remained high at 99.95%, supporting customer retention efforts
- Churn in Mobile Postpaid and Fixed Broadband segments exceeded overall churn, risking $8–12K annual revenue loss
