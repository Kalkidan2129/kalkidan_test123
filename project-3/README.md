
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created a comprehensive Power BI dashboard to analyze the effect of pilot awareness on wildlife strike incidents. It compares outcomes between well-informed and uninformed pilots to assess damage costs and passenger safety. The data was sourced and modeled using SQL Server and visualized with interactive slicers and KPIs.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and cause costly damages. There is a need to understand how pilot awareness and timely warnings can reduce these incidents and improve flight safety. Airlines require actionable insights to mitigate risks and control associated costs.

---

## Objective

- Analyze the impact of pilot warnings on wildlife strike damage costs
- Visualize key metrics such as total deaths and injuries related to strikes
- Create an interactive dashboard for stakeholders to explore data by time and region

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling

---

## Project Workflow

- Load wildlife strike data into SQL Server and create data mart
- Import and model data in Power BI
- Design dashboard with slicers for time, flight phase, and region
- Create KPI cards for total deaths and injuries
- Develop measures to compare warned vs. uninformed pilot outcomes

---

## Key Insights

- Well-informed pilots experience lower wildlife strike damage costs
- Timely warnings correlate with fewer injuries and fatalities
- Certain regions and flight phases have higher wildlife strike risks
- Interactive slicers enable detailed temporal and regional analysis
