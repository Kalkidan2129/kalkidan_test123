
# Walmart Store Sales

## Project Overview

This project involved creating a comprehensive Power BI dashboard to analyze sales data from 45 Walmart stores between 2010 and 2012. It focused on visualizing year-over-year sales trends and the impact of economic conditions during the post-2008 recession period. The dashboard incorporated advanced DAX measures and custom data modeling techniques.

---

## Business Problem

Walmart stores, like other discount retailers, faced sales challenges emerging from the 2008 recession. Understanding sales trends and economic impacts was critical to strategize recovery and growth. The project aimed to provide actionable insights to improve sales performance during this recovery phase.

---

## Objective

- Analyze year-over-year sales trends across multiple Walmart stores
- Incorporate economic factors like unemployment and holiday periods into sales analysis
- Develop an interactive Power BI dashboard for clear visualization and decision support

---

## Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
- Excel
- CSV Data Files

---

## Project Workflow

- Import Walmart sales CSV data into Power BI Desktop
- Create calendar and refresh date tables using DAX
- Establish relationships between sales and calendar tables
- Add conditional columns for holiday flags and unemployment rates
- Develop DAX measures to calculate total sales, average sales, and year-over-year growth

---

## Key Insights

- Sales showed variability influenced by holiday weeks and economic conditions
- Unemployment rates impacted store sales performance during the analyzed period
- Year-over-year sales growth metrics highlighted recovery trends post-recession
- Custom sorting and calendar integration enhanced temporal analysis accuracy
