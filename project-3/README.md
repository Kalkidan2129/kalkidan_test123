
# Air Traffic Control Awareness and Wildlife Strikes

## Project Overview

This project created an interactive Power BI dashboard to analyze the impact of pilot awareness on wildlife strike incidents. It compares outcomes between well-informed and uninformed pilots to assess damage costs and passenger safety. The solution integrates SQL Server data modeling with KPI visualizations to support aviation safety decisions.

---

## Business Problem

Wildlife strikes pose significant risks to aircraft safety and cause costly damages. Airlines need actionable insights on how pilot awareness affects incident severity and repair costs. Improving warning systems can enhance passenger safety and reduce financial losses.

---

## Objective

- Analyze the effect of pilot warnings on wildlife strike damage and safety
- Create a data mart and interactive dashboard for aviation stakeholders
- Develop KPIs to monitor warning effectiveness and cost impact

---

## Tools & Technologies

- Power BI
- SQL Server
- Microsoft Office
- SQL scripting
- Data modeling
- Dashboard reporting

---

## Project Workflow

- Load and prepare wildlife strike data using SQL Server
- Import and model data within Power BI
- Create slicers for time, flight phase, and region filtering
- Develop KPI cards for pilot warnings, damage, and costs
- Design and finalize interactive dashboard for insights

---

## Key Insights

- Well-informed pilots correlate with reduced damage costs from wildlife strikes
- Increased pilot warnings lead to improved passenger safety outcomes
- Year-over-year trends show changes in warning effectiveness and incident rates
- Damage and repair costs vary significantly by flight phase and region
